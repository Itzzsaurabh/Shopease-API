// features

What it does

01
JWT Authentication
Stateless token-based auth. Every request is verified by JwtAuthFilter before reaching the controller. BCrypt password hashing — passwords never stored in plain text.
02
Product Management
Full CRUD operations on products. Case-insensitive search by name. Stock tracking. Category filtering. All data persisted in MySQL via Spring Data JPA.
03
Shopping Cart
Add items, auto-merge duplicates (quantity increases), remove individual items. Cart is user-specific — identified by JWT token, no session storage needed.
04
Order Processing
Place order from cart in one call. Auto-calculates total. Saves price at purchase time permanently. Cart auto-clears after order. Full order history available.
05
Global Exception Handling + Swagger Docs
All errors return clean JSON responses via @RestControllerAdvice — no ugly stack traces. All 13 endpoints are auto-documented and testable at /swagger-ui/index.html with zero extra configuration.

OrderService.java
@Service
@Transactional
public Order placeOrder(String email) {
  // 1. find user from JWT email
  User user = userRepo.findByEmail(email);

  // 2. get all cart items
  List<CartItem> items =
    cartRepo.findByUser(user);

  // 3. calculate total
  BigDecimal total = items.stream()
    .map(i -> i.getPrice()
    .multiply(i.getQty()))
    .reduce(ZERO, BigDecimal::add);

  // 4. save order atomically
  Order order = Order.builder()
    .user(user)
    .totalAmount(total)
    .status("PENDING")
    .build();

  // 5. clear cart
  cartRepo.deleteByUser(user);
  return orderRepo.save(order);
}
